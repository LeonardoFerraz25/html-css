Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)
Nesta frase temos um termo em negrito usando a tag <strong> (semântica)

Italico / Ênfase

Nesta frase temos um termo em italico usando a tag <i> (não semântico)
Nesta frase temos um termo em italico usando a tag <em> (semântico)
Nesta frase temos um

termo em italico
usando a tag <address> (semantico para dar enfase no endereço)

Sublinhado

Nesta frase temos um termo sublinhado usando a tag <u> (não semântico)
Nesta frase temos um termo sublinhado usando a tag <ins> (semântico)

Marcador de texto

Podemos criar tambem um marcador de texto usando a tag <mark> e trocar a cor do marcador usando a css style=background-color:(e a cor que quiser)

Texto pequeno

podemos criar um texto pequeno usando a tag <small>

Texto deletado

podemos riscar um texto com a tag <del>

Texto sobrescrito

Podemos iseriri potencias com do tipo x20+5 usando a tag <sup>

Texto subsrito

Podemos inserir textos desse tipo H2O usando a tag <sub>
